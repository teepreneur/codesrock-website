# CodesRock: Logic Card System Specification

This document defines the UI/UX standards for the physical logic cards used in the CodesRock ecosystem.

## 1. Physical Specifications
*   **Dimensions:** 63.5 mm x 88.9 mm (2.5" x 3.5").
*   **Corner Radius:** 3.5 mm.
*   **Finish:** Matte UV Coating (to minimize glare for optical sensors).
*   **Back-of-Card:** Includes a unique QR code or NFC tag identifier for Rockbot reading.

## 2. UI Design System (Palette)
We use a "Vibrant Unplugged" palette to appeal to children while maintaining brand consistency.

| Command | Primary Color | Hex Code | Icon Description |
| :--- | :--- | :--- | :--- |
| **Forward** | Action Orange | `#FF7340` | Upward arrow. |
| **Turn Left** | Primary Blue | `#46C5D5` | Curved arrow (left). |
| **Turn Right** | Growth Yellow | `#FDC82F` | Curved arrow (right). |
| **Loop** | Creative Pink | `#EC4899` | Circular arrow. |
| **Special/Start**| Deep Purple | `#5D3B98` | Play/Stars/etc. |

## 3. Core Icon Set
1.  **Forward (Move):** A centered, upward-pointing arrow with a slight geometric "glow" at the tip.
2.  **Turn Left/Right:** A 90-degree curved arrow indicating rotation.
3.  **Repeat (Loop):** A circular arrow with a blank box in the center for "quantity" stickers or chalk.
4.  **Start (System):** A "Play" triangle within a glowing orange circle.

## 4. Visual Accents
*   **The Border:** A subtle, grayscale Kente geometric pattern along the left and right edges.
*   **The Rocky Seal:** A small, simplified silhouette of Rocky's head—characterized by two bulbous antennae—in the bottom-right corner for brand authenticity.
*   **High Contrast:** All icons must be white on a dark primary color background to ensure readability in various lighting conditions.
