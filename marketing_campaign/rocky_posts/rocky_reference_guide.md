# Rocky "The Logic Star" — Visual Reference Guide
## 🔒 USE THIS FILE EVERY TIME YOU GENERATE A ROCKY IMAGE

This document is the **single source of truth** for Rocky's visual appearance. Reference it before every image generation to ensure brand consistency across all posts.

---

## 1. Character Anatomy (MANDATORY)

| Feature | Specification | Notes |
|---------|--------------|-------|
| **Body Shape** | Rounded, compact alien body. Slightly pear-shaped (wider at bottom). Approx 3 heads tall. | Pixar-lite proportions |
| **Body Color** | Vibrant orange `#FF7340` | Must be saturated, never muted/brownish |
| **Body Texture** | High-quality 3D "soft-touch" matte finish | Like a premium vinyl toy |
| **Heritage Dots** | Subtle darker-orange dot pattern on arms, belly, and back | Inspired by the logo monsters. ALWAYS include. |
| **Belly/Chest Patch** | Rounded lime-green `#7CFC00` patch on the belly/chest area | Visible from the front |
| **Lime-Green Accents** | Small green accent marks on cheeks, shoulders, and knees | These add character detail — don't omit |
| **Antennae** | Two bulbous-tipped antennae rising from the top of the head | Organic, slightly curved like bobbing springs |
| **Antennae Tips** | Bright glowing lime-green orbs | Must always glow — this is Rocky's signature |
| **Eyes** | Large, round, warm brown eyes with visible reflective highlights | Expressive and friendly. NEVER angry. |
| **Mouth** | Small, friendly. Smiling or open with excitement. | Match expression to the post's mood |
| **Hands** | 4 fingers (3 + thumb). Orange. Slightly oversized for expressiveness. | Important for gestures (wave, rock-on 🤘, pointing) |
| **Feet** | Small, rounded. No toes. Same orange as body. | Slightly stubby |

### Key Poses Library
- **Wave** (Post 1): Right hand raised, open palm wave. Left arm relaxed at side.
- **Pointing/Holding** (Posts 2, 3): One hand holds an object, other hand gestures.
- **Rock On 🤘** (Post 6): Right hand raised in "rock on" sign. Left fist pumped.
- **Hands on Hips** (Post 5): Confident, Superman-style stance. Chest forward.
- **Group Pose** (Post 4): Standing center, slightly forward of squad members.

## 2. Speech Bubble Style (MANDATORY)

| Property | Specification |
|----------|--------------|
| **Shape** | Rounded cloud/thought bubble with soft tail pointing to Rocky |
| **Material** | Semi-transparent 3D glassmorphic (frosted glass effect) |
| **Border** | Subtle glowing white/cyan edge |
| **Background** | Translucent — the scene behind should be faintly visible through it |
| **Font** | Clean, bold, white sans-serif text. NOT hand-drawn/comic font. |
| **Font Color** | White `#FFFFFF` for main text. Emojis in full color. |
| **Text Size** | Large enough to read clearly at Instagram thumbnail size (~600px) |
| **Internal Effect** | Subtle digital code rain or sparkle particles inside the glass |

> ⚠️ DO NOT use multi-color gradient text inside speech bubbles consistently. Reserve gradient/colored text ONLY for special announcement posts (e.g., UNICEF).

Use this for Posts 1–5 and any narrative/story-driven content.

| Element | Description |

| **Coding Elements** | Subtle code symbols, logic gates, and gear motifs woven into nature |

## 4. Classroom Environment (Secondary Background)

Use for real-world grounding posts (UNICEF, school partnerships, testimonials).

| Element | Description |
|---------|-------------|
| **Setting** | Bright, modern, well-lit classroom |
| **Floor** | Light wooden floor or warm-toned tiles |
| **Walls** | Clean white/cream walls. Whiteboard optional (blank or with basic doodles) |
| **Props** | Coding cards on the floor/table, CodesRock Activity Book visible |
| **Lighting** | High-key, warm, natural light from windows |
| **🚫 NO SCREENS** | **CRITICAL: No tablets, laptops, monitors, or any screen devices visible.** This violates the core brand message. |

- [ ] Heritage dot pattern visible on body
- [ ] Lime-green belly patch present
- [ ] Green accent marks on cheeks/shoulders
- [ ] Both antennae present with glowing green tips
- [ ] Eyes are large, warm brown, friendly

- [ ] CodesRock logo bottom-right
