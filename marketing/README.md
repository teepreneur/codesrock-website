# CodesRock Marketing Folder

This directory contains all the assets, strategies, and templates for the CodesRock marketing campaigns.

## Contents
- `campaign/`: Contains overarching strategy, audience breakdown, and the 3-month roadmap.
- `social_media/`: Contains drafted posts and visual guidelines for LinkedIn, Instagram, and Facebook.
- `email/`: Contains automated email sequences for parents (B2C) and school outreach (B2B).
- `assets/`: Designated folder for finalized images, graphics, and video clips based on the brand guidelines (currently empty pending design team).
