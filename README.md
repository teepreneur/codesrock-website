# CodesRock React Website

This is a full React conversion of the CodesRock website, built from the original HTML generated by aura.build.

## Features

- **Fully responsive design** - Works on all screen sizes
- **Modern React architecture** - Component-based structure
- **Tailwind CSS styling** - With custom animations and colors
- **Lucide React icons** - Beautiful, consistent icons throughout
- **Smooth scrolling** - Enhanced user experience with smooth navigation
- **Interactive elements** - Hover effects, animations, and transitions

## Project Structure

```
src/
├── components/
│   ├── BackgroundElements.jsx  # Decorative background blobs and icons
│   ├── Navigation.jsx           # Top navigation bar
│   ├── Hero.jsx                 # Hero section with animated characters
│   ├── SocialProof.jsx          # Partner/client logos
│   ├── WhyScreenFree.jsx        # Why Screen-Free section
│   ├── KitSection.jsx           # What's in the kit
│   ├── Impact.jsx               # Impact statistics
│   ├── Testimonials.jsx         # Customer testimonials
│   ├── About.jsx                # About section
│   ├── ContactForm.jsx          # Contact form
│   ├── Footer.jsx               # Footer with links
│   └── FloatingCTA.jsx          # WhatsApp button and mobile CTA
├── App.jsx                      # Main app component
├── index.css                    # Global styles and Tailwind directives
└── main.jsx                     # React entry point
```

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Navigate to the project directory:
```bash
cd codesrock-react
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

### Building for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```

## Components Overview

### BackgroundElements
Decorative background blobs and floating icons that create visual interest.

### Navigation
Sticky navigation bar with logo, menu links, and CTA button.

### Hero
Main hero section featuring:
- Animated badge
- Large headline with gradient text
- Call-to-action buttons
- Three animated character illustrations

### SocialProof
Displays trusted partner logos and institutions.

### WhyScreenFree
Explains the benefits of screen-free learning with:
- Feature highlights
- Image showcase

### KitSection
Grid of 6 cards showing what's included in the CodesRock kit.

### Impact
Statistics section with purple background showing program impact.

### Testimonials
Two-column grid of customer testimonials with quotes.

### About
Information about the founders and mission.

### ContactForm
Interactive form for booking demos with:
- Name, phone, email fields
- Role selection (School Rep / Parent)
- Submit button

### Footer
Footer with logo, social media links, and legal links.

### FloatingCTA
- WhatsApp floating button
- Mobile sticky CTA bar
